HOWTO RUN THIS POC : il faut obligatoirement servir la POC avec un serveur (car module js à loader) → le plus simple est de npm run dev (cf. ./build_and_run.sh).

  • Utilisation d'une map et un marqueur via vue.js (sans vue-leaflet) ; quand on bouge le marqueur, ses coordonnées affichées sont mises à jour dans les résultats
  • La hiérarchie de composants est : Map > MyContainer > { MyMap>MyMarker + MyResults }
  • Premier point notable = MyMarker attend que la map leaflet soit créée pour finir son initialisation (vu qu'il a besoin de s'addTo à la map)
  • Second point notable = la façon dont MyMarker communique sa position, pour que celle-ci soit affichée par MyResult (façon assez horrible à mes yeux)
