HOWTO RUN THIS POC : il faut obligatoirement servir la POC avec un serveur (car module js à loader) → le plus simple est de npm run dev (cf. ./build_and_run.sh).

Ce que fait cette POC = un viewer leaflet avec vue.js :

  • lit la position de la carte et du marqueur dans l'URL, par exemple : à Montpellier , à Bordeaux, sur la localisation par défaut, ...
  • ouvre une map leaflet + un marqueur selon ce qu'indique l'URL
  • (pas de sous-composants utilisés ici : vue n'a que peu d'intérêt pour cette POC, mais elle montre au moins l'utilisation de leaflet via vue)
