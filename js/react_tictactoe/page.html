État atteint en terme d'UX : on peut cliquer sur le board pour afficher ou masquer une croix.

État atteint en terme de techniques react :

  • squelette général = intégration dans la page du composant parent : ReactDOM.render(, document.getElementById('root'));
  • fonction composant = composants qui ont un simple render ne dépendant que de leurs props (ici Square et Game)
  • composant à état Board, qui a un state pour garder une trace de la partie en cours
  • le composant à état Board contrôle la fonction composant Square
  • notamment, lorsqu'on clique sur un Square, le handler utilisé est celui du composant parent Board
  • c'est par ce moyen que le composant fils Square peut mettre à jour l'état du composant parent Board
